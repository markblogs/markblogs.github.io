执行 shell 脚本首先要查看 用户是否具有执行文件(需要执行的shell文件) 文件权限

输出环境变量

环境变量配置在 /etc/profile

在 shell 脚本中可以使用 $ + 变量名 进行环境变量查看和使用

用户变量

shell脚本中用户自定义的变量，用户变量长度不可以超过 20 位

用户变量在使用时 $ + 变量名 进行使用

命令替换

从命令中提取信息，并将其赋值给变量。

有两种方式可以将命令赋值给变量

  • 反引号字符 (`)
  • $() 格式

重定向输入和输出

重定向有两种

  • 将命令输出重定向到另一个位置：输出重定向

    • > 重定向命令，会覆盖文件原有内容

    • >> 在原有文件进行追加

  • < 可以将文件重定向到命令输入：输入重定向

执行数学运算

  • expr 命令

  • 使用方框号

  • 浮点值运算 bc 命令

退出脚本

shell 中 每运行一个命令都使用退出状态码 （exit status）

Linux 专门提供了变量 $? 变量来保存上一个命令的退出状态码。

exit 命令 允许在脚本结束时指定一个状态码
