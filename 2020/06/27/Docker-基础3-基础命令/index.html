Docker_基础3_基础命令

Posted on 2020-06-27 Edited on 2021-04-23 In 容器技术

帮助命令

  • 查看版本：
    • docker version
  • docker信息：
    • docker info
  • docker命令帮助：
    • docker –help

镜像命令

  • 列出本机所有镜像：
    • docker images [options]
      • -a：列出本地所有的镜像（含中间映像层）
      • -q：只显示镜像ID
      • –digests：显示镜像的摘要信息
      • –no-trunc：显示完整的镜像信息
  • 仓库中检索镜像：
    • docker search [options]
      • –no-trunc：显示完整镜像描述
      • -s：列出收藏数不小于指定值的镜像
      • –automated：只列出 automated build（自动构建）类型的镜像
  • 拉取镜像：
    • docker pull ImageName:tag
  • 移除镜像：
    • docker rmi ImageName
    • 移除所有镜像：docker rmi -f $(docker images -qa)

容器命令

  • 新建并启动容器

    • docker run [options] image [command]

      • –name：容器名称
      • -d：后台运行，并返回容器id
      • -i：交互模式运行容器，通常与 -t 同时使用
      • -t：为容器重新分配一个伪输入终断
      • -P：随机端口映射
      • -p：指定端口映射

  • 查看所有正在运行的容器

    • docker ps

  • 退出容器

    • exit：停止并退出容器
    • ctrl+p+q：容器不停止退出

  • 启动容器

    • docker start 容器id或者容器名

  • 重启容器

    • docker restart 容器id或者容器名

  • 停止容器

    • docker stop 容器id或者容器名

  • 强制停止容器

    • docker kill 容器id或者容器名

  • 删除已经停止的容器

    • docker rm 容器id
    • docker rm $(docker ps -aq)

  • 查看容器日志

    • docker logs -f -t –tail 容器id
      • -t：加入时间戳
      • -f：跟随最新的日志打印
      • –tail 数字：显示最后多少条

  • 查看容器内运行的进程

    • docker top 容器id

  • 查看容器内部细节（）

    • docker inspect 容器id

  • 进入正在运行的容器并以命令交互

    • docker exec -it 容器id bashShell （容器中打开新的终端，不会的启动新的进程）
    • 重新进入：docker attch ring器id（直接进入容器启动命令的终端，不会启动新的进程）

  • 从容器内拷贝文件到主机上

    • docker cp 容器id:容器内路径 目的主机路径
